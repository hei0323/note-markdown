# Markdown 标题

## 1、用符号 = 和 - 表示标题等级

书写格式如下：

```
一级大标题
=========

二级大标题
---------
```

预览效果图如下：
一级大标题
=========

二级大标题
---------



## 2、用符号 \* 表示标题等级

书写格式如下：
```
# 一级大标题

## 二级大标题

### 三级中标题

#### 四级中标题

##### 五级小标题

###### 六级小标题

```

预览效果图如下：
# 一级大标题

## 二级大标题

### 三级中标题

#### 四级中标题

##### 五级小标题

###### 六级小标题
