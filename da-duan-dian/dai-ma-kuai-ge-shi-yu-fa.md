## MarkDown 代码块的表示

#### ~~1.使用 4个空格 或 1个制表符表示~~

* ##### 书写格式如下：

  &lt;?php  
   echo 'hellow wold!';  
   ?&gt;

* ##### 预览效果如下：

  &lt;?php   
   echo 'hello wold!';  
  ?&gt;

#### 2.使用 \`\` 号表示

* ##### 书写格式如下：

  ```php
   `<?php echo 'hello wold';?>`
  ```
* ##### 预览效果如下：

  `<?php echo 'hello wold';?>`

#### 3.使用 ``````` ``` ````表示

* ##### 书写格式如下：

  ```php
      ```php
      <?php
      echo phpinfo();
      ?>
      ```
  ```
* ##### 预览效果如下：

  ```
  <?php
  echo phpinfo();
  ?>
  ```
